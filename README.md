SQL project analyzing **customer behavior, retention, and segmentation** using the **AdventureWorksDW2025** data warehouse.

This project demonstrates how SQL can generate business insights such as:

- Customer Cohorts  
- Retention Rate  
- Customer Churn Detection  
- Customer Lifetime Value  
- Revenue Segmentation  
- RFM Customer Segmentation  

---

# Dataset

**Database:** AdventureWorksDW2025  

Main tables used:

- `FactInternetSales`
- `DimCustomer`

---

# Analysis Overview

### 1️⃣ Cohort Analysis

Groups customers by their **first purchase month** to analyze retention behavior across time.

### 2️⃣ Customer Retention Rate

Calculates the percentage of customers who **return after their first purchase**.

### 3️⃣ Customer Activity Status

Classifies customers as **Active or Churned** based on their last purchase date.

### 4️⃣ Customer Lifetime Value (CLV)

Measures the **total revenue generated by each customer**.

### 5️⃣ Customer Revenue Segmentation

Segments customers into **revenue tiers** using SQL window functions.

### 6️⃣ Customer RFM Segmentation

Segments customers based on **RFM metrics**:

| Metric | Description |
|------|-------------|
| Recency | Days since last purchase |
| Frequency | Number of orders |
| Monetary | Total spending |

Example customer segments:

- Champions
- Loyal Customers
- Potential Loyalists
- At Risk
- Lost Customers

1. Load **AdventureWorksDW2025** database  
2. Open **SQL Server Management Studio** (or preferred SQL environment)  
3. Run scripts in order:
01 → 02 → 03 → 04 → 05 → 06


Each script produces analytical outputs that can be used for **reporting or dashboards**.
