# Customer Retention Analytics (SQL Server)

This project analyzes customer purchasing behavior using SQL to measure **retention, churn, and customer lifetime value (CLV)**.

The analysis is built using the **AdventureWorksDW2022** dataset in Microsoft SQL Server and demonstrates common analytics techniques used in real business environments.

---

# Project Objective

The goal of this project is to understand customer lifecycle behavior by answering questions such as:

- When do customers make their first purchase?
- How many customers return after their first purchase?
- What percentage of customers are retained each month?
- Which customers are likely to churn?
- What is the total value generated by each customer?

These insights help businesses improve customer retention and revenue growth.

---

# Dataset

Dataset used: **AdventureWorksDW2022**

Main tables used:

- FactInternetSales – Sales transactions
- DimDate – Date dimension
- DimCustomer – Customer information
- DimProduct – Product information

The dataset follows a **Star Schema**, commonly used in data warehouses.

# Analysis Performed

### Cohort Analysis
Groups customers by their **first purchase month** and tracks their activity over time.

### Retention Rate
Calculates the percentage of customers who return after their first purchase.

Retention Formula:

```
Retention Rate = Active Customers in Month N / Cohort Size
```

### Churn Detection
Identifies customers who stop purchasing for a long period of time using purchase gaps.

### Customer Lifetime Value (CLV)
Calculates the total revenue generated by each customer.

### Revenue Segmentation
Segments customers based on their total revenue contribution.
