# Customer Retention Analytics (SQL Project)

This project analyzes **customer behavior and retention patterns** using SQL.  
It demonstrates how transactional sales data can be transformed into **business insights such as retention rate, churn detection, customer lifetime value, and customer segmentation**.

The project is built using **SQL Server and T-SQL**, organized into modular SQL scripts that simulate a simple **data transformation pipeline**.

---

# Dataset

This project uses the **AdventureWorks2025** sample database from Microsoft SQL Server.

AdventureWorks is a **sample transactional database** that represents a fictional manufacturing company and includes tables for sales, customers, products, and orders.

Key tables used in this project include:

```
FactInternetSales
DimCustomer
DimDate
```

These tables provide the necessary data to perform **customer behavior and retention analysis**.

# Data Flow

This project follows a **SQL transformation pipeline** that converts raw transactional data into analytical datasets.

```
Raw Sales Data
      │
      ▼
Customer Purchase History
      │
      ▼
Cohort Identification
      │
      ▼
Retention & Activity Analysis
      │
      ▼
Customer Lifetime Value
      │
      ▼
Customer Segmentation (Revenue + RFM)
```

Each SQL script represents a **transformation step** in the analysis workflow.

---

# Analysis Overview

### 1. Cohort Analysis
Groups customers based on their **first purchase month** to track behavior over time.

### 2. Customer Retention Rate
Calculates the percentage of customers who **return after their first purchase**.

### 3. Customer Activity Status
Classifies customers as:

- Active  
- Inactive  
- Churned  

based on the **time since their last purchase**.

### 4. Customer Lifetime Value (CLV)
Estimates the **total revenue generated by each customer** across their lifetime.

### 5. Revenue Segmentation
Segments customers into **High, Medium, and Low revenue groups**.

### 6. RFM Segmentation
Segments customers using the **RFM model**:

| Metric | Description |
|------|------|
| Recency | How recently the customer purchased |
| Frequency | How often the customer purchases |
| Monetary | How much the customer spends |
